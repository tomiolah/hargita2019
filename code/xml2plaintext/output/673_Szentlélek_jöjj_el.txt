673. Szentlélek jöjj el

v1
1, Szentlélek jöjj el égi fénnyel, 
Álmából minden bűnöst költs fel, 
A hívők szívét töltsd be hittel! 
Isten Szentlelke, jöjj!

v2
2. Zendülj meg zúgó égi szélként, 
Űzd el a bűnök átkos éjét, 
Ragyogtasd rajtunk Jézus fényét! 
Isten Szentlelke, jöjj!

v3
3. Permetezz ránk, mint égi harmat, 
Ébreszd fel újra azt, ki lankad, 
Jöjjön a mára még szebb holnap! 
Isten Szentlelke, jöjj

v4
4. Lángoljon újra tiszta fényed, 
Erődtől béna, holt feléled, 
Te vagy az el nem múló élet! 
Isten Szentlelke, jöjj!
