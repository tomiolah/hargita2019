507. Mindenem az Úrnak adom

v1
1. Mindenem az Úrnak adom, Minden az Övé. 
Szívemet csak Néki adom, Minden az Övé. 
Néki szóljon dobbanása, Töltse el szent öröm, hála, 
Légyen a képmása rajta! Minden az Övé.

v2
2. Mindenem az Úrnak adom, Minden az Övé. 
Hangomat is Néki adom, Minden az Övé.
Másokat Ő Hozzá hívni, Róla bátran vallást tenni, 
Tetteit mind elbeszélni, Minden az Övé.

v3
3. Mindenem az Úrnak adom, Minden az Övé. 
Vágyaim Őnéki adom, Minden az Övé. 
Szívből híven Őt szolgálni, Máson mindig könyörülni,
Lelkeket Hozzá vezetni, Minden az Övé.

v4
4. Mindenem az Úrnak adom, Minden az Övé. 
Életem csak Néki adom, Minden az Övé. 
Vágyam, buzgó törekvésem: Érte élni minden percben, 
Áldani Őt majd a mennyben, Minden az Övé.
