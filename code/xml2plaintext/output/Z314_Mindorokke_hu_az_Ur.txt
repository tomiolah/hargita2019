Z314 Mindorokke hu az Ur

v1
1. /:Mindörökké hû az Úr,:/
Mindig áldom én az Õ nevét, 
Mindörökké hû az Úr.

v2
2. /:Minden jóval áld az Úr,:/
Mindig áldom én az Õ nevét, 
Minden jóval áld az Úr.

v3
3. /:Kísértésben óv az Úr,:/
Mindig áldom én az Õ nevét, 
Kísértésben óv az Úr.

v4
4. /:Lelkével betölt az Úr,:/
Mindig áldom én az Õ nevét, 
Lelkével betölt az Úr.
