Z273 Lent a Via Dolorosan

v1
Lent a Via Dolorosán, 
Jeruzsálemben történt; 
Egy férfi vitte súlyos keresztjét. 
Körülállta õt a nép, az embert, 
Akit halálra ítélt.

v2
Jézus testén vérzõ sebek, 
Korbácsütés nyomai, 
Koronája vétkeink tövisei. 
Minden lépést értünk tett, 
Így bûneinktõl szabadított meg.

v3
Lent a Via Dolorosán, 
Minden fájdalom útján 
Isten Fia feláldozta önmagát. 
Végtelen nagy szeretettel 
Vállalta érted, értem, 
Lent a Via Dolorosán, 
A halált, ó, a halált.
