Z329 Nekem az elet Krisztus

v1
1. Nékem az élet Krisztuskövetés,
Krisztus az út, a halál nyereség,
Örömmel telt szívem nem fél,
Mert Lelke bennem él,
Nékem az élet Krisztuskövetés!

v2
2. Nékem az élet Krisztuskövetés!
Utaimon az Õ hatalma véd.
Lelke által felvehetem 
Szellemi fegyverem,
Nékem az élet Krisztuskövetés!

v3
3. Nékem az élet Krisztuskövetés!
Legyõzhetem a sátán erejét.
Elõttem drága cél lebeg, 
Mely a mennybe vezet.
Nékem az élet Krisztuskövetés!

v4
4. Nékem az élet Krisztuskövetés!
Szerelme elvet minden szenvedélyt.
Azóta boldogan mondom,
Te vagy Szabadítóm,
Nékem az élet Krisztuskövetés!
