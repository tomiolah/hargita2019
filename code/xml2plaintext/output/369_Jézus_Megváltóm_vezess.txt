369. Jézus Megváltóm, vezess

v1
1. Jézus, Megváltóm, vezess 
Biztos révbe engemet!
Tomboló viharon át 
Győzz le minden akadályt!
Hogyha Véled indulok, 
Örök célomhoz jutok.

v2
2. Bőszült tenger törhet rám, 
Rémíthet bár vadul hullám; 
Parancsszódra csendesül, 
Néked enged, egyedül. 
Tűnik minden fergeteg, 
Hogyha int erős kezed.

v3
3. Majd ha utam végezem, 
S örök révbe érkezem, 
Uram, légy ott is velem, 
Szüntesd minden félelmem!
Szelíd hangon bátoríts, 
Dicsfényedben részesíts!
